New Features for March 2016

Now supporting more year models! 2007-2016

A new FP3 (part number 66007) supports the following bikes:
  • 2007-2010 Softail
  • 2007-2011 Dyna
  • 2007-2013 Touring
  • 2007-2013 Sportster
The original FP3 (part number 66005) continues to support:
  • 2011-2016 Softail
  • 2012-2016 Dyna
  • 2014-2016 Touring
  • 2014-2016 Sportster
Same app for both part numbers.
We also squeezed in some bug fixes and general improvements.
