New Features for February

Instantaneous fuel economy

View Sensor Data can show the current fuel economy in “real time”.

Average fuel economy

View Sensor Data also accumulates fuel usage and miles traveled to give an average fuel economy value. When this is displayed in the large center gauge, a small “reset” button will appear at the bottom of the gauge for resetting the accumulated fuel and miles. Fuel use is logged by the FP3 so the FP3 must remain plugged into the bike to use this feature. The app does not need to be running except to view the value.

Power

View Sensor Data can show the (approximate) power currently being generated by the engine.

Torque

FP3 uses the power results above to calculate and display a torque value in View Sensor Data.

Personalized VIN

For use in demo mode: you can type in your bike’s VIN to see exactly what maps are available for your bike.

The System Info page now includes the current odometer reading as well as the odometer reading the last time the ECU was flashed.
